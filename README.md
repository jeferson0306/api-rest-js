### PROJECT EXECUTION
#### Step 1
Clone project
```shell script
$ git clone https://github.com/jeferson0306/api-rest-js.git
```

#### Step 2
Enter root folder
```shell script
$ cd api-rest-js
```

#### Step 3
```shell script
$ cd src
```

#### Step 4
Run maven commands to build the project
```shell script
$ cd mvn clean install
```

#### Step 5
Run .jar
```shell script
java -jar target/salesforce-customer-bulk-batch.jar
```
